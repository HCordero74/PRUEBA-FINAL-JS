# Conversor de Monedas
Último desafío de JavaScript consiste en un conversor que posee tres opciones de monedas, se ingresa el monto y lo transforma de acuerdo a la moneda que uno elija, realizando la transformación y presentándolo en pantalla. Por último también se presenta el gráfico con los valores de los últimos 10 días de la moneda seleccionada.
